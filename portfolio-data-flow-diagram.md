## Key Principles

1. **UI is Presentation Only** - No business logic in components
2. **Services Call Edge Functions** - All data fetching in service layer
3. **Graceful Fallback** - Mock data if edge functions fail
4. **Defense in Depth** - Multiple safety checks prevent crashes
5. **Demo Mode First** - Works without wallet connection
6. **Wallet Switching** - Data updates on wallet change
7. **Console Logging** - Always know data source (REAL vs MOCK)
