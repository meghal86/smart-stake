- Edge CDN for anonymous users (5 min TTL)
- Redis for hot data (1 hour TTL)
- Client-side with React Query (1 min stale time)

## Integration Points

### API Endpoints
- `GET /api/hunter/opportunities` - Fetch opportunities
- `GET /api/guardian/summary` - Guardian trust scores
- `GET /api/eligibility/preview` - Eligibility status
- `POST /api/hunter/save` - Save opportunity
- `POST /api/hunter/report` - Report opportunity

### State Management
- Local state for UI (filters, modals)
- React Query for server state
- URL params for shareable state

### Event Tracking
- `feed_view` - Page view
- `filter_change` - Filter applied
- `card_impression` - Card viewed (0.1% sampling)
- `card_click` - Card clicked (100% sampling)
- `save` - Opportunity saved
- `cta_click` - CTA button clicked
