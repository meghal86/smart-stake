Testing Checklist

When Demo Mode is ON:

  • Blue banner appears: "Demo Mode ‚Äî Showing simulated opportunities"
  • Wallet chip shows "Demo Wallet 0xd8dA...6045" with DEMO badge
  • 5 sample opportunities load instantly
  • No API calls in Network tab
  • Data is static (Ethereum Staking, LayerZero, etc.)

When Demo Mode is OFF:

  • Banner disappears completely
  • Wallet chip shows real wallet address (or "No Wallet Connected")
  • API call to /api/hunter/opportunities
  • Real opportunities from backend/blockchain
  • Infinite scroll works with pagination
  • Real-time updates every 60 seconds

Switching Between Modes:

  • Toggle in profile dropdown updates immediately
  • No page reload required
  • Data switches seamlessly
  • No crashes or errors
  • Scroll position maintained

Manual Testing Steps

  1. Open Hunter page at http://localhost:8088/hunter
  2. Verify Demo Mode is ON (blue banner visible)
  3. Check wallet chip shows "Demo Wallet" with DEMO badge
  4. Verify 5 sample opportunities are displayed
  5. Open DevTools Network tab
  6. Confirm no API calls to /api/hunter/opportunities
  7. Click profile dropdown ‚Üí Toggle "Demo Mode" OFF
  8. Verify banner disappears
  9. Check wallet chip updates to real wallet (or "No Wallet Connected")
  10. Verify API call appears in Network tab
  11. Check opportunities update to real data
  12. Scroll down to test infinite scroll (live mode only)
  13. Toggle Demo Mode back ON
  14. Verify everything switches back to demo state
