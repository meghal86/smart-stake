Debugging Steps

  1. Open browser DevTools (F12)
  2. Go to Console tab
  3. Click on buttons and watch for console messages
  4. If no messages appear, the onClick handlers are not working
  5. Check Elements tab to see if event listeners are attached

Common Issues

  • Event Propagation: Check if events are being stopped
  • Disabled State: Verify buttons are not disabled when they should be enabled
  • asChild Prop: Ensure props are forwarded correctly to motion.button
  • Tooltip Wrapper: Check if tooltip wrapper is blocking clicks
